# 热点转选题知识

原始热点不能直接入选题，必须完成一次转换：

- 模型发布 -> 普通人/运营/创作者能用它完成什么任务？
- 产品更新 -> 新功能解决了哪个旧痛点？
- 热搜争议 -> 能不能做成工具对比、成本拆解或风险判断？
- 广告主线索 -> 产品卖点里哪些可以实测，哪些不能夸大？
- 截图文字 -> 提取工具名、功能点、用户痛点和可验证任务。

不要强行迁移：

- Coding、底层模型、基础设施、融资、收购、政策、安全、诈骗类新闻，如果没有直接对应的普通人工具入口或内容生产任务，就不要进入候选。
- 不能只因为“可以写提示词”“可以做设计助理”“可以做商业判断”就把跨赛道热点包装成当前账号选题。
- 对视觉/视频/设计/电商图定位的账号，优先找能直接产出图片、视频、封面、海报、PPT、网页、商品图、素材或修图结果的热点。

高价值选题通常包含：

- 真实工具入口
- 真实使用场景
- 可截图的操作过程
- 成功结果和失败边界
- 读者下一步能照着做的行动
